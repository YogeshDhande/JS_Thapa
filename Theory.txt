                  ----------- Variable -------------  
Variable:-A Variable is a container that holds a value.It has a name and can be used to store and manipulate data in program.

Values:-A value is piece of information that a program can work with.It can be number, text, true/false, etc.

Rules and Best Practies:-
 -Variable name must start with an letter, or an underscore(_), or an Dollar sign($).
 -Variable cannot contain spaces.
 -Variable cannot be as same as reserved keyword such as if, const, new, this.....
 -Variable are case sensitive.
 -Variable can be as long as you need
 -By convention, javascript Variable names are written in camelCase.

Example:-
    var my_firstName ="Java";               //Valid
    var _mylastName$ ="Script";             //valid
    var 123myAge = "25"                     //InValid
    var $cityName = "Netscape"              //Valid
    var my@Email = "javascript@gmail.com"   //Invalid

  let reverse = "";
  for(let char= str.length - 1; char>= 0; char--){
    reverse = reverse + str[char];
  }
  return reverse;
}
